# VR-Game-Absurdity-

We are part of the MAVERiC Studio Garage Startup Cohort 8 and we are trying to create a VR game where the objective is to escape the cursed hotel.

The game will be on VR Chat and the game involves the main character stuck in the top floor of the hotel, and they need to solve the various 
puzzles around the hotel in order to progress through the floors of the hotel.
